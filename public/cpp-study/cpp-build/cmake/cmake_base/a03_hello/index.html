    代码

    CMakeLists.txt

    cmake_minimum_required(VERSION 3.26)

project(cmake-cpp-hello CXX)

aux_source_directory(. SRCS_MAIN)
add_executable(${PROJECT_NAME} ${SRCS_MAIN})

    main.cpp

    #include <iostream>
int main(){
    std::cout << "Hello World" << std::endl;
    return 0;
}

    如何编译并运行

    无论是 linux 还是 windows , 都可以使用如下命令进行构建 

    # windows 为 md build
mkdir build
cd build
cmake ..
cmake --build .
