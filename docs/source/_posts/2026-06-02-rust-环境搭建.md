# rust 安装

## windows

## 下载

https://rust-lang.org/zh-CN/learn/get-started/

https://static.rust-lang.org/rustup/dist/x86_64-pc-windows-msvc/rustup-init.exe

![](images/2026-06-02-rust-环境搭建/001.png)


## 环境变量设置

**CARGO_HOME**

`cargo` 配置, 缓存, 依赖下载路径

**RUSTUP_HOME**

`rust` 编译器, 工具链安装位置

**RUSTUP_DIST_SERVER**

编译器下载镜像

```
https://mirrors.tuna.tsinghua.edu.cn/rustup
```


**RUSTUP_UPDATE_ROOT**

`rustup` 自身更新地址

```
https://mirrors.tuna.tsinghua.edu.cn/rustup/rustup
```

## 设置依赖镜像地址

`%USERPROFILE%\.cargo\config.toml`

```toml
[source.crates-io]
replace-with = 'tuna'

[source.tuna]
registry = "https://mirrors.tuna.tsinghua.edu.cn/git/crates.io-index.git"

[net]
git-fetch-with-cli = true
```

## 安装

此时双击 `rustup-init.exe` , 然后直接回车即可

然后将 `cargo/bin` 目录添加到 `PATH`
