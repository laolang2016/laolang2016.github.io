后端代码修改

  1. 修改 ruoyi-admin/src/main/resources/application.yml,修改端口号和 redis 地址
  2. 修改 ruoyi-admin/src/main/resources/application-druid.yml,修改 mysql 链接
  3. 修改 ruoyi-admin/src/main/resources/logback.xml, 修改日志路径
